Preguntas Frecuentes

¿Cuál es el propósito de este proyecto?

Este trabajo tiene como objetivo facilitar el aprendizaje en la ciberseguridad mediante el uso de IAs existentes pre-entrenadas.

¿Que IA genera nuestra información?

La IA utilizada como referencia es ChatGPT 4.0 Mini, pero utiliza solo información proporcionada por nosotros o los gestores de contenido

¿Qué herramienta se ha utilizado para entrenar la IA?

La IA que responde a las consultas de los usuarios ha sido entrenada con la herramienta Stack.ai.

¿Cómo funciona el modo original?

Este modo cuenta con una IA pre-entrenada con la finalidad de aportarte todos los conocimientos que necesites sobre ciberseguridad.

¿Cómo funciona el modo entrenamiento?

En este modo, la IA parte del modo original con la diferencia de estar abierta a recibir información nueva por parte de los usuarios.

¿Cuántos roles hay y qué hace cada uno?

Hay 4 roles:
1. Administrador: se encarga del correcto funcionamiento de todo y puede aceptar y/o rechazar solicitudes de usuarios verificados y no verificados.
2. Gestor de contenido: este rol sirve para administrar y manejar el modo de aprendizaje en el que hay una base de datos abierta en la que los usuarios pueden añadir archivos y el propio gestor, con un web-scraper, añadir información a la base de datos de forma automática.
3. Usuario verificado: se trata de un usuario que puede realizar consultas a nuestro agente IA y puede subir archivos a la base de datos que tendrán que ser verificados. Además, puede realizar solicitudes de rol de gestor de contenido.
4. Usuario no verificado: es un usuario que sólo puede realizar consultas al agente IA. Para tener más permisos tiene que realizar una solicitud al admin.

¿Qué es un web scrapper?

Un web scraper es un programa que extrae datos de páginas web de forma automática.Es la tecnología que usamos para la búsqueda automática de archivos en el modo aprendizaje.
